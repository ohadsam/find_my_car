# 🚗 FindMyCar - מצא את הרכב שלך

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://ohadsam.github.io/find_my_car/)
[![PWA Ready](https://img.shields.io/badge/PWA-Ready-blue)](https://ohadsam.github.io/find_my_car/)

## 🚀 שימוש מהיר

1. פתח את [FindMyCar](https://ohadsam.github.io/find_my_car/) בדפדפן הנייד
2. לחץ **"שמור חניה עכשיו"** → האפליקציה שומרת את מיקום ה-GPS
3. הוסף תמונה / הקלטה / תיאור (אופציונלי)
4. כשתרצה לחזור לרכב - לחץ **"נווט"**

## 📲 התקנה על המסך הראשי

### Android (Chrome)
1. פתח את האתר ב-Chrome
2. הופע באנר "הוסף למסך הבית" → לחץ **"התקן"**
3. לחלופין: תפריט ⋮ → **"הוסף למסך הבית"**

### iOS (Safari)
1. פתח את האתר ב-Safari
2. לחץ על כפתור השיתוף 📤
3. גלול מטה → **"הוסף למסך הבית"**

## 🛠️ טכנולוגיות

- **HTML5 / CSS3 / Vanilla JavaScript** - ללא frameworks
- **Leaflet.js** - מפות אינטראקטיביות
- **OpenStreetMap** - מפות חינמיות
- **Nominatim** - המרת קואורדינטות לכתובות
- **Service Worker** - תמיכה offline
- **localStorage** - אחסון מקומי
- **MediaDevices API** - מצלמה ומיקרופון
- **Geolocation API** - GPS
- **Web Share API** - שיתוף מובנה
